# data/

  Synthetic geoscience sample data for loading into Neo4j.

  > All data is synthetic - no real proprietary data included.

  ## Files

  | File | Description |
  |---|---|
  | `wells_sample.csv` | 15 synthetic wells with UWI, depth, status, operator |
    | `surveys_sample.csv` | 6 synthetic seismic surveys (2D/3D) |
    | `formations_sample.csv` | 10 geological formations with age and lithology |
